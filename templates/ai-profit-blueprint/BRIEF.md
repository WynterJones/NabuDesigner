## Sales architecture (section order — the page the copy agent builds)

1. **Sticky nav** — logo, one-liner, ghost "Get Free Access" → `#optin`.
2. **Hero (opt-in + VSL)** — eyebrow ("FREE 5-DAY MINI-COURSE"), big-promise H1,
   direct-response subhead, the **email opt-in form** (email + `Get Free Access`
   button-image), trust microcopy ("No cost. No fluff. Unsubscribe anytime."),
   a **VSL play card** (host thumbnail + play badge → YouTube modal). Hero has a
   signature background loop + particle field. This is the squeeze.
3. **Agitation strip** — "If you're still doing X by hand…" pain bullets with
   generated icons; count-up stat ("hours/week lost").
4. **Future-pace / promise** — what life looks like after the 5 days; scene image.
5. **What you'll learn (the 5 days)** — 5 module cards, each a day with a generated
   icon + a fascination one-liner ("Day 3: the 20-minute system that…").
6. **Host authority** — Marcus Vale cutout, credibility bullets, VSL replay CTA.
7. **Fascination bullet stack** — Brunson-style "Secret #N" curiosity bullets with
   check icons; mid-page opt-in CTA band.
8. **Proof** — fictional testimonials (avatar images), results stat strip, "as-seen" shape logos.
9. **What you get / value stack** — it's free, but stack the value (lessons,
   swipe file, prompt pack, community) with a gold "$0 today" framing + module render.
10. **FAQ** — accordion (is it really free, who it's for, time required, etc.).
11. **Final opt-in (urgency)** — restated big promise, the email form again,
    reason-why urgency (cohort/limited bonus), guarantee-of-value microcopy, host P.S.
12. **Footer** — fictional disclaimer, nav, no real claims.

## CTA rule
The page is **lead-gen**: the primary action is the **email opt-in form** (hero +
mid + final). The big image-buttons are the form's submit (`Get Free Access`).
Keep a sticky mini opt-in / scroll-to-form behavior. No price, no checkout.
