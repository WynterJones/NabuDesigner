# Local Lake Fishing Course

Fictional static landing-page template for a `$28` local lake fishing video course.

Open `index.html` directly in a browser. No build step is required.

Asset notes:

- `hero-local-lake.webp` is an ImageMagick-generated graphic fallback because runtime AI image generation failed during setup.
- `cta-main.png` and `cta-main-hover.png` are image-backed CTA button states.
- `favicon.png` is generated locally.
